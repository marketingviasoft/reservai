# ReservAI

Aplicacao interna para gestao de equipamentos, reservas, check-in e check-out.

> **Importante:** a documentacao completa de arquitetura, regras de negocio, estado atual do projeto e roadmap esta em [HANDOFF.md](./HANDOFF.md).

## Rodando localmente

Este repositorio usa `pnpm` como gerenciador de pacotes.

### Instalar dependencias

```bash
corepack pnpm install
```

### Subir ambiente de desenvolvimento

```bash
corepack pnpm dev
```

### Comandos uteis

```bash
corepack pnpm check
corepack pnpm test
corepack pnpm build
```
